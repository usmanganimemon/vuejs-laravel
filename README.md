# ecommerce-front

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
Please provide set VUE_APP_SERVER_URL in .env file

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
